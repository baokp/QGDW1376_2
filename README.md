# QGDW1376_2
Q/GDW1376.2 Protocol message parsing tool

## 简介
14年写的Q/GDW1376.2规约解析工具，使用C语言编写，资源占用很小，可在嵌入式单片机中运行。

## 开发环境
- MinGW32/gcc
- qt 4.8.7

## 使用方法
- 请查看GDW1376_2.h
- 截图
